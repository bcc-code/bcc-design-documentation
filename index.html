BCC Logos

Description

Technical documentation and guides for devs to facilitate the correct use of the logo of the BCC association, joint initiatives and local congregations on digital platforms (websites, applications etc.).

How to use it

PNG

https://design.bcc.no/assets/[organisation-name]/logo_[organisation-name]_[color]_[size].png
[size]: 64/32

SVG

https://design.bcc.no/assets/[organisation-name]/logo_[organisation-name]_[color].svg
[orgisation-name]:

i.e:

  • bcc-eiker
  • bcc-bergen
  • bcc-drammen-sande

[color]: primary-green

[color]: white
